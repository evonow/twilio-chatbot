Здравствуйте!

Хочу представить завершенный проект **Ask GroupFund** — интеллектуальную систему чат-бота на базе искусственного интеллекта для поддержки клиентов через SMS (Twilio) и веб-интерфейс.

## Что было разработано:

Система использует технологию RAG (Retrieval Augmented Generation) для ответов на вопросы на основе базы знаний из прошлых взаимодействий с клиентами, документации и материалов поддержки. Основные возможности:

- **Многоканальная поддержка**: SMS через Twilio и веб-интерфейс
- **Управление базой знаний**: веб-GUI для загрузки и обработки обучающих данных
- **Поддержка множества форматов**: .eml, .docx, .pdf, .csv, .txt, .json, .xml
- **Контроль доступа на основе ролей**: Admin, Internal, Sales Rep, Customer
- **Интеграция с GitLab**: автоматическое извлечение коммитов, README и релизных заметок
- **История разговоров**: поддержание контекста между сообщениями
- **Атрибуция источников**: показывает, какие документы использовались для генерации ответов
- **Анализ FAQ**: автоматическое извлечение часто задаваемых вопросов

## Стоимость проекта:

Общая стоимость разработки составила **$54,000** (720 часов работы по ставке $75/час).
